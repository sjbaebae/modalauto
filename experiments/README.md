# Experiments

Experimental side projects and support datasets live here so they do not clutter the repository root.

- `matmul/`: default autoresearch reference experiment. It owns its generated `journal/` and `worktrees/` directories.
- `openai_hide_and_seek/`: vendored OpenAI hide-and-seek MAE environment used by related experiments.
- `randomized_uncertain_social_preferences/`: social-preference environment experiments that build on the OpenAI MAE environment package.
